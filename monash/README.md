# Monash

Write shell scripts in a monadic way.

## Example

```bash
$ ./monash "fortune >> echo Meh >>= cowsay"
Don't Worry, Be Happy.
		-- Meher Baba
 _____
< Meh >
 -----
        \   ^__^
         \  (oo)\_______
            (__)\       )\/\
                ||----w |
                ||     ||
```
